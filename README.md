# Autosar Spec spider

#### 介绍
AUTOSAR官网spec爬虫

#### 软件架构
使用scrapy框架实现autosar文档的爬取，


#### 安装教程

1. 安装scrapy:  
   pip install scrapy
1. 安装Beautiful Soup:  
   pip install BeautifulSoup4


#### 使用说明

1. scrapy crawl Autosar --nolog
